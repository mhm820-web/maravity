# 단어 시험 프로그램

영어 단어 학습을 위한 웹 기반 시험 프로그램입니다.

## 기능

- 6개 레벨별 단어 시험 (각 1000개 단어)
- 문제 수 선택 (5/10/20/30/50/100/1000문제)
- 즉시 채점 및 결과 확인
- 인쇄용 단어장/시험지 생성

## 로컬 실행

```bash
pip install -r requirements.txt
python app.py
```

브라우저에서 http://localhost:5000 접속

## 배포

이 프로젝트는 Render, Railway, Heroku 등에서 배포 가능합니다.
